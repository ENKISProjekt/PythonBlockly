Implementing Algorithms

An algorithm is the conceptual solution to a problem. An algorithm only comes to life in a programming language.

The conversion of an algorithm into a programming language is called implementation.


A program therefore implements an algorithm. There are different programming languages that can be chosen to implement an algorithm. 

Visual Programming Language: uses interactive graphical user interface (GUI) to help users program. Users can drag and drop the parts that often resembles blocks or puzzle pieces that can be connected to make a program. Examples include: Snap! and Scratch. We will use Blockly as an example in the next chapters.

Text-based Programming Language: uses texts, or codes, to program. Users have to memorize certain syntaxes or grammar specific to each programming language, but a well trained programmer can write more versatile codes in text-based programming language faster than in a visual programming language. Examples include: Python, Java, and C. We will use Python as an example in the next chapters.

In the following sections, you will build your first programs by using both visual programming language (Blockly) and text-based programming language (Python), that can be written in one language and translated to the other automatically. Ultimately, the goal is to help you familiarize yourself with Python, which is one of the programming languages that are used most widely for implementing machine learning programs.

After implementing* the program, you will want to run the program and see the results. Among many, Print is one of the easiest way to see the results. Let us explore how this works, by clicking through the menus below!
