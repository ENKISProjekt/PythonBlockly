Data Types and Operations


Now that we can print out the results in a way that we want, it is time to do some calculations. Well, computers are built to compute at the first place, so it would be logical for us to head next to basic operations.

Similarly in math, calculations in programming are done with variables and operators, where variables are used to store data and operators define a certain manipulation of the data. Observe the following example.

a = 2
b = 3
s = a + b

In above code, a, b, s are variables. In Python, values are stored in variables by using the '=' operator.  '+' is another operator, which in above case, yields the sum of the two integers 2 and 3, which is subsequently stored in the variable 's' by the '=' operator. In the end, 's' will have the value of 5.

x and y are the variables in the code and are used to store a real number (5/2) and a text ("Hello World") respectively. 

"Hello", "World", 5, and 2 are called data, which have different data types. 

/, +, *  are the operators in the code. Note that they function somewhat differently depending on data types.

Notice how 5/2 and 5/2.0 gave different results?

This is because 2 and 2.0 are treated differently in Python. Although both intuitive, “+” in “Hello”+”World” and 1+2 are also treated differently. This is because Python has different data types.

1.1 is an integer and often marked with int.
2.“Hello” is a string (text) and often marked with str.
3.2.0 is a float (real number) and marked with float.
An integer divided by an integer will also yield an integer. If you wish to get a mathematically more precise result, you need to use floats.
Strings can be added to each other to form a longer text, which is a process called concatenation. They can also be multiplied by an integer to repeat the string to yield a longer string, which is a process called scalar multiplication.
You don't need to memorize how each operation functions for each combination of data types. The important thing is to understand that there are different data types and operations may function differently to different data types.
