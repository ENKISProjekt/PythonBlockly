Variables and Operations

Ein Rechner ist gebaut, um zu rechnen. For a computer program that implements an algorithm therefore involves a lot of calculations, or in other words, mathematical operations. Let us continue our discussion after looking at a simple algorithm that calculates and prints out the sum of two numbers. 

print("Hello, World!")

In above code, a, b, s are variables and '+', '=' are operators. '=' operator, also called the assignment operator, stores the value on the right hand side to the variable on the left hand side. Therefore, after the line a=2, the value 2 will be stored in the variable a. '+' operator is simple: it adds two values to yield their sum.

Operators are symbols or keywords in programming languages that perform operations on operands, such as arithmetic, logical, or comparison tasks (e.g., +, &&, ==).

Variables are named storage locations in memory that hold data values which can be manipulated and referenced throughout a program.

In this chapter, we will only learn arithmetic operators (and of course, the '=' operator.) You will learn other types of operators in the later sections.



Data Types

Can variables only store numbers? No, they can store many different types of data. In programming languages, data are said to have types and different data types behave differently with different operators. For example, 2+3 will yield 5, but "Hello"+"World" will yield "HelloWorld" Although both are quite intuitive, it is evident that the '+' operator means two different things for numbers and texts. The following table explains some important data types in Python. Familiarize yourself with the data types, as texts will be called 'strings(str)' and numbers will be called either 'integers(int)' or 'floats' from now on.

Data Type Call Name Description Example
int Integer Represents whole numbers, positive or negative, without decimals. x = 42
float Floating-Point Number Represents decimal numbers (floating-point numbers). y = 3.14
str String Represents text, enclosed within single or double quotes. name = 'Alice'
bool Boolean Represents truth values, either True or False. is_valid = True
